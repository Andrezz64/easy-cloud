# S3 File Manager

## Navigation

- Click a bucket from the S3 list or Dashboard to open the file browser
- Use the **breadcrumb bar** to navigate between folders
- Click folder names to enter them, or use breadcrumbs to go back

## Uploading Files

### Simple Upload
- Click the **Upload** button in the header
- Select one or more files
- Files ≤ 10MB use single PUT upload
- Files > 10MB automatically use **multipart upload** (10MB chunks)

### Create Empty File
- Right-click in empty area → **New File**
- Enter a filename (e.g., `config.json`)
- The file is created with the correct Content-Type based on extension

## File Actions

### Via action buttons (visible on each row)
- 👁 **Preview** — Opens side panel with file preview
- ↓ **Download** — Save file locally via dialog
- ⋮ **More** — Dropdown with additional actions

### Via right-click context menu
Right-click on a file:
- Preview, Edit, Download
- Share Link (presigned URL)
- Copy To, Move To
- Rename
- Delete

Right-click on empty area:
- Upload Files
- New Folder
- New File
- Refresh
- Properties

## Code Editor

For text-based files (.html, .css, .js, .json, .yaml, .py, .md, etc.):

1. Right-click → **Edit** (or Preview → click **Edit** button)
2. Full-screen editor opens with syntax highlighting
3. Edit the content
4. **Ctrl+S** saves directly to S3
5. Status bar shows line count, file size, language
6. Orange dot (●) indicates unsaved changes

Supported languages: YAML, JSON, HTML, CSS, JavaScript, TypeScript, Python, Markdown, XML, and more.

## Presigned URLs (Share Links)

Generate a temporary download link:

1. Right-click a file → **Share Link**
2. Select expiration: 15min, 1h, 6h, 12h, 1 day, 3 days, 7 days
3. Click **Generate Link**
4. Copy the URL — anyone with the link can download the file until it expires

## Bucket Properties

Use the tabs at the top of the bucket view:

- **Objects** — File browser (default)
- **Permissions** — View and edit the bucket policy (JSON)
- **Properties** — Versioning status, encryption, region
- **Tags** — Manage bucket tags (key-value pairs)

## Copy Bucket URL

Click the 📋 icon next to the bucket name to copy the S3 endpoint URL:
```
https://my-bucket.s3.us-east-1.amazonaws.com
```
