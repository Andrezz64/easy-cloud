## About the project

Easy Cloud is a **free, open source desktop app** for managing AWS resources visually. No terminal gymnastics, no confusing console, no data leaving your machine.

- 🔒 **100% local** — credentials stay on your device
- 🚀 **Native performance** — Tauri + Rust, not Electron
- 💸 **Zero cost** — the tool is free, forever
- 🌍 **Community-driven** — built by people who believe cloud should be accessible to everyone
