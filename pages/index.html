S3 File Manager

A full file browser that feels native — but for the cloud.

Browse & NavigateBreadcrumb navigation, click folders, back/forward — like your OS file explorer.
Smart UploadAuto multipart for files >10MB. Chunked, with progress bar. Correct Content-Type detection.
Instant PreviewText, code, images previewed inline. 50+ extensions recognized even when S3 metadata is wrong.
In-App Code EditorSyntax highlighting for YAML, JSON, HTML, CSS, JS, Python, etc. Ctrl+S saves directly to S3.
Presigned Share LinksGenerate temporary URLs (15min to 7 days). Anyone can download — no AWS credentials needed.
Copy / Move / RenameRight-click any file. Copy between buckets, move (copy+delete), batch operations.
Bucket ConfigurationEdit bucket policy (JSON), toggle versioning, view encryption, manage tags — all in tabs.
Context MenusRight-click everywhere. Empty area: upload, new folder, new file. On files: preview, edit, share, delete.

CloudFormation

Infrastructure as Code — deploy, monitor, manage stacks visually.

YAML Editor + ValidationSyntax highlighted editor. Validate button catches errors before you deploy.
Visual BuilderConfigure S3, SQS, SNS resources via forms. Generates valid YAML automatically.
One-Click DeployName your stack and go. Real-time tracker shows every resource being created.
Drift DetectionDetect if resources were modified outside CF. See exactly what drifted per resource.
Change SetsPreview updates BEFORE applying. See adds, modifies, removes, and destructive replacements.
Template ViewView deployed template. Load back to editor. Get AWS Calculator cost estimation link.
Live Operation TrackerDeploy/Update/Delete tracked in real-time. Progress bar, per-resource status, elapsed time.
Cross-Stack DependenciesView exports and imports between stacks. Understand your infrastructure dependency graph.

Billing & Cost Explorer

Understand your AWS spend without the console.

Monthly SummaryCurrent month total, last month, and end-of-month forecast. Delta percentage indicator.
Cost Over Time ChartBar chart with adaptive granularity. Daily for short ranges, monthly for long periods.
Top ServicesProportional bars showing which services cost the most. Visual and instant.
Usage BreakdownPer-service usage with rates: 150GB x $0.023/GB = $3.46. Free tier items tagged green.
Period Filter7 days, 1 month, 3 months, 6 months, 1 year, or custom month range picker.
Smart 2h CachePer-account cache. Switch accounts instantly from cache. Only manual Refresh costs $0.01.

Built-in Code Editor

Edit files directly on S3 — like VS Code for the cloud.

Syntax HighlightingYAML, JSON, HTML, CSS, JavaScript, TypeScript, Python, Markdown, XML and more.
Ctrl+S Saves to S3Press save and the file uploads with correct Content-Type. No temp files, no friction.
Full-Screen EditorImmersive experience. Line numbers, status bar, language badge, unsaved changes indicator.
Perfect for Static HostingEdit index.html, style.css, app.js live on S3. Changes are instant — no deploy pipeline.
Zero Friction OpenRight-click file > Edit. Or click Edit in preview panel. Two clicks to start editing.
Create New FilesRight-click > New File. Name it (e.g. config.json) and start editing immediately.

Security & Privacy

Your data, your machine, your rules.

100% Local ExecutionNo proxy servers. API calls go directly from your machine to AWS. Nothing in between.
Credentials Stay HomeStored in localStorage on your device. Never transmitted anywhere except AWS endpoints.
Zero TelemetryNo analytics, no tracking, no error reporting to third parties. Nothing phones home.
Rust BackendCompiled native binary via Tauri. No Electron bloat. Memory-safe, fast, auditable.
Fully Open SourceEvery line is public. Inspect, audit, fork. AGPL-3.0 guarantees it stays open.
Lightweight~8MB installer. Under 50MB RAM. No background processes when the app is closed.
